美化终端

需要以下Linux/Mac软件包：git、zsh

参考：程序员DJW（美化并增强Mac终端）

更改默认shell

切换默认shell为zsh:

chsh -s /bin/zsh	#切换到默认为zsh

安装oh-my-zsh

# 下载oh-my-zsh
git clone https://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh

# 创建一个新的配置文件
cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc

安装插件

命令高亮插件

顾名思义就是打对命令会显示为绿色，命令不对显示为红色

# 下载命令高亮插件
git clone https://github.com/zsh-users/zsh-syntax-highlighting ~/.zsh/zsh-syntax-highlighting

# 配置插件生效
echo '# 命令高亮\nsource ~/.zsh/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh' >> ~/.zshrc

命令提示插件

会将历史执行过的命令进行灰色显示，方便复用命令

# 下载命令提示插件
git clone https://github.com/zsh-users/zsh-autosuggestions ~/.zsh/zsh-autosuggestions

# 配置插件生效
echo '# 命令提示\nsource ~/.zsh/zsh-autosuggestions/zsh-autosuggestions.zsh' >> ~/.zshrc

最后执行：

zsh

工作管理

工作管理即是，将前台应用放置于

程序放入后台执行

  • 一行待执行命令结尾&，回车执行放入后台

    #按照速度排序mirrorlist（比较耗时）
Immortal-Arch# rankmirrors -n 6 /etc/pacman.d/mirrorlist.backup > /etc/pacman.d/mirrorlist &
[1] 63059
Immortal-Arch# 
[1]  + done       rankmirrors -n 6 /etc/pacman.d/mirrorlist.backup > /etc/pacman.d/mirrorlist
    • 其中[1]工作号，63059进程号
    • [1] + done +号说明最近，加入的工作号为[1]命令已经执行完成

  • 执行过程中Ctrl + Z，使前台命令暂停放于后台

    Immortal-Arch# top

查看后台放入的命令

jobs命令可查看当前工作区暂停或后台运行的程序。

选项 含义
-l（L 的小写） 列出进程的 PID 号。
-n 只列出上次发出通知后改变了状态的进程。
-p 只列出进程的 PID 号。
-r 只列出运行中的进程。
-s 只列出已停止的进程。 
849 ◯  jobs 
[1]    suspended (signal)  top
[2]  - suspended (signal)  top
[3]  + suspended (signal)  top
Immortal-Arch ॐ  ~:
850 ◯  jobs -l
[1]    63720 suspended (signal)  top
[2]  - 64078 suspended (signal)  top
[3]  + 64112 suspended (signal)  top
  • +号代表默认工作，恢复后台就是此进程
  • -号代表第二个加入的工作

后台命令脱离终端运行

系统更新时如果退出终端或者退出登录就会使更新中的程序中断，这时就需要脱离终端运行。

三种方法：

  1. 系统启动时执行/etc/rc.local文件
  2. 定时任务
  3. nohup

nohub命令的使用：

 nohup rankmirrors -n 6 ./mirrorlist.backup > ./mirrorlist &
Immortal-Arch ॐ  ~:
893 ◯  nohup rankmirrors -n 6 ./mirrorlist.backup > ./mirrorlist &
[1] 69294
nohup: ignoring input and redirecting stderr to stdout                                                         
Immortal-Arch ॐ  ~:
894 ◯  jobs
[1]  + running    nohup rankmirrors -n 6 ./mirrorlist.backup > ./mirrorlist
Immortal-Arch ॐ  ~:
895 ◯  
[1]  + 69294 done       nohup rankmirrors -n 6 ./mirrorlist.backup > ./mirrorlist
Immortal-Arch ॐ  ~:
895 ◯  jobs

注意：

  1. 图形化命令窗口点击关闭会停止当前工作台的所有后台进程，一定要执行完点关闭。
  2. 远程登录直接可以退出xshell等远程连接软件，后台程序不会退出

管道符

cd aaa && ls        #前面的执行成功后面的才可以执行
cd aaa || ls        #前面的执行不成功，后面的才执行，f反之后不执行
